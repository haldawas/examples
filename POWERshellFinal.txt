Create a powershell script that will gather information about the host it is executed on and write the results to a file.

	-survey	[outfile]
		Computername
		Date/Time
		OS Version
		List of all processes,sorted (or separated, or grouped) by Session ID 
		List of all open sockets

	-hashdir <dir> [outfile]	
		Get contents of directory
		Generate and save a hash of each file in the directory

	-custom
		Make the script perform an action of your own design.
